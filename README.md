# Many Interesting Things for Aspiring Engineers

This is a repository that contains Python notebooks from the MIT Beaver Works Summer Institute Girls Who Want to Learn Many Interesting Things! Fall 2024 course.

## Key Learnings

- Performed extensive data analysis using Python libraries such as NumPy, Pandas, and Scikit-learn
- Completed advanced coursework in machine learning and deep learning, applying concepts to 7 collaborative projects addressing real-world healthcare challenges like diagnostic tools and predictive analytics
